# PixelDraw — Pixel Art Canvas

A pixel art drawing app with color palette, draw/erase/fill/pick tools, undo/redo, grid toggle, PNG export, and a local gallery of saved artworks.

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)
![HTML Canvas](https://img.shields.io/badge/HTML_Canvas-E34F26?style=flat)

## Features

- **Drawing Tools** — Pen, eraser, flood fill, color picker
- **60-Color Palette** — Full spectrum with recent colors tracking
- **Grid Sizes** — 16x16 to 64x64 canvas options
- **Undo / Redo** — Full history with 50-step undo
- **Export PNG** — Download your art as a scaled PNG
- **Gallery** — Save artworks locally and reload them
- **Grid Toggle** — Show/hide pixel grid overlay
- **Touch Support** — Works on mobile and tablet
- **Keyboard Shortcuts** — D=Draw, E=Erase, F=Fill, P=Pick, G=Grid

## Getting Started

```bash
git clone https://github.com/TamerAli-0/pixel-draw.git
cd pixel-draw
# Open index.html in your browser
```
